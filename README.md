# Personal Finance Dashboard

A Python project that tracks cash, credit cards, investments, net worth, utilization, and credit score history.

## Features

- View full financial summary
- Track checking, savings, and investment balances
- Track credit card balances and limits
- Calculate credit utilization
- Calculate estimated net worth
- Add credit score updates
- Simulate how payments affect utilization
- Saves data in a JSON file

## How to Run

```bash
python finance_dashboard.py
```

## Skills Used

- Python
- Functions
- Dictionaries
- JSON file handling
- Loops
- User input
- Basic financial calculations
